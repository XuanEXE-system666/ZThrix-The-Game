# ZThrix-The-Game
这是一个用 TurboWarp 制作的经典俄罗斯方块游戏。具有完整的游戏逻辑、金币系统和主题系统等。

## 玩法说明

- 使用左右方向键移动方块
- 使用空格键旋转方块
- 使用上方向键直接落到底
- 使用下方向键加速下落
- 使用c键进行方块暂存
- 当方块堆叠到顶部时游戏结束

## 如何运行

### 在线运行
你可以通过以下链接直接在浏览器中运行
- [项目发布页面](https://xuanexe-system666.github.io/ZThrix-The-Game/) 

### 离线代码
1. 下载本仓库中的 `ZThrix.sb3` 文件。
2. 打开[TurboWarp 在线版编辑器](https://turbowarp.org/editor)或下载并打开[离线编辑器](https://desktop.turbowarp.org/)
3. 从文件中加载 `ZThrix.sb3` 并点击绿旗开始。

## 项目文件
